【1】命令
	[1]cd命令
		cd 绝对路径
		cd 相对路径
		
		cd  /etc
		或者
		cd ../../etc
		
		cd ~ 或cd   回到家目录下
		cd -        回到上一次操作的目录下
		cd /        切换到根目录下
		cd .        切换到当前目录下
		cd ..       切换到上一级目录下
		
	[2]pwd
		显示当前的路径
		
		
	[3]touch 创建文件
		touch  文件名
		如果文件存在，更新文件的时间戳
		如果文件不存在，创建文件
		
		
		touch  文件1  文件2  文件3  ...
		同时创建多个文件
		
	[4]mkdir 创建目录
		mkdir  目录名
		//用来创建目录
		
		mkdir -p  1/2/3  
		创建具备层级关系的目录
		
		mkdir -m  0777 list
		创建一个具备0777权限的目录
	
	[5]chmod  修改权限
		
		chmod 权限 文件/目录
		
		比如：
			chmod 0777 hello/  
			将hello目录的权限修改为0777
			
			chmod 0441 2.c
			将2.c的文件的权限修改为0441
